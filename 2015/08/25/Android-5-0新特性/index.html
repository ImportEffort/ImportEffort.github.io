View的高度与阴影

官网介绍：https://developer.android.com/intl/zh-tw/training/material/shadows-clipping.html

  • 注意这个内容是在5.0之后的出现的，所以只有在5.0后的手机上才能有效果
  • View增加了高度的概念，高度大的View会覆盖在高度小的View之上，并带有阴影效果：

  • View高度有2个因素组成：

    • View高度 = elevation + translationZ
    • elevation表示view的高度，高度越大，阴影越大，可以在xml中直接使用属性， 也可以在代码中使用： view.setEvelvation();
    • 在xml指定： android:elevation="10dp
    • transtionZ 属性表示view在Z方向移动的距离，一般用于属性动画中，translationZ也会影响View的高度，但是对阴影没有效，同时增加了改变translationZ的属性动画;在xml中指定：android:translationZ="10dp"

    • 动画效果，通过指定translationZ的方法来实现：

      ObjectAnimator animator = ObjectAnimator.ofFloat(view,"translationZ",0,100);
    animator.setDuration(2000);

    animator.start();

View的轮廓与裁剪(在Android5.1以及以上才有效果)

官网介绍：https://developer.android.com/intl/zh-tw/training/material/shadows-clipping.html

5.1之后：View增加了轮廓概念，轮廓会影响阴影的效果和裁剪的形状。

  • View的轮廓默认是依据于background的，也就是轮廓的形状依赖于我们对其指定的背景的形状来决定，这里需要注意所有的view的大小都是一个矩形。

    android:outlineProvider="background"//按照指定的背景的形状来描绘阴影的大小

  • 有其他3个取值：bounds,none,paddingBounds，可以通过outlineProvider属性更改:
    ​ 

    android:outlineProvider="none"//如果设置为none 即使view有背景形状和高度，也不会显示阴影
android:outlineProvider="bounds"//按照View的大小来描绘阴影
android:outlineProvider="paddingBounds" //与bounds类似，阴影会稍微向右平移一点

  • 可以通过setOutlineProvider()方法自定义轮廓:

    首先不要为控件指定一个背景形状shape

    <TextView
 android:layout_centerInParent="true"
 android:id="@+id/tv1"
 android:background="@mipmap/dog"
 android:gravity="center"
 android:elevation="20dp"
 android:text="textView1"
 android:textSize="30sp"
 android:layout_width="200dp"
 android:layout_height="200dp" />

    然后再代码中调用：

    tv1.setOutlineProvider(new ViewOutlineProvider() {
      @Override
      public void getOutline(View view, Outline outline) {
          //参数轮廓的边缘的坐标，坐标值是相对于view本身左上角的坐标
          //1. 设置轮廓为圆形，默认为直径为矩形的宽度
          outline.setOval(0,0,view.getWidth(),view.getHeight());
          //2. 设置轮廓为圆角矩形，最后一个参数为矩形圆角的弧度
          //outline.setRoundRect(0,0,view.getWidth(),view.getHeight(),200);
          //3. 设置轮廓为矩形  前两个参数为轮廓的左边和上边的起始坐标，后两个参数为宽度，和高度
          //outline.setRect(0,0,view.getWidth()-100,view.getHeight()-100);
          //4.自定义轮廓的形状，这里定义了一个三角形
          //Path path = new Path();
          //path.moveTo(view.getWidth()/2,0);
          //path.lineTo(0,view.getHeight());
          //path.lineTo(view.getWidth(),view.getHeight());
          //outline.setConvexPath(path);
      }
  });
  //根据轮廓裁剪view的形状 目前不支持自定义轮廓的形状的裁剪
  //tv1.setClipToOutline(true);

所以有了这个特性以后，我们就可以轻松的设置一个圆形头像了，并不需要背景图片是个圆形，只需要控制轮廓就可以了。

Palette 的使用

使用Palette可以让我们从一张图片中拾取颜色，将拾取到的颜色赋予ActionBar，StatusBar以及背景色可以让界面色调实现统一

  • 简单使用：
  1. 添加依赖：使用Palette需要添加以下依赖： compile 'com.android.support:palette-v7:23.0.0+'

  2. 使用Palette提供的API，传入Bitmap即可获取Palette对象,以下是同步和异步使用方式:

    1. 同步获取，需要在子线程中使用

      Palette palette = Palette.from(drawable.getBitmap()).generate();

    2. 异步获取，可以在主线程中使用

      Palette.from(drawable.getBitmap()).generate(new Palette.PaletteAsyncListener() {
    @Override
    public void onGenerated(Palette palette) {
        //...
    }
});

  3. 通过回调接口传回的Palette对象即可拿到bitmap中的相应颜色，拿到的颜色分为：

    vibrant      -  有活力的颜色
lightVibrant -  有活力的亮色
darkVibrant  -  有活力的暗色
muted        -  柔和暗淡的颜色
lightMuted   -  柔和的亮色
darkMuted    -  柔和的暗色

  4. 获取方法分为两种：第一种方法如下，返回值可以直接设置给相应的view对象：

     Bitmap bitmap = BitmapFactory.decodeResource(getResources(), R.mipmap.bird);
//异步获取不需要手动开启线程
Palette.from(bitmap).generate(new Palette.PaletteAsyncListener() {
    @Override
    public void onGenerated(Palette palette) {
        //vibrant      -  有活力的颜色
        int vibrantColor = palette.getVibrantColor(getResources().getColor(R.color.bg_deep_orange));
        //lightVibrant -  有活力的亮色
        int lightVibrantColor = palette.getLightVibrantColor(getResources().getColor(R.color.bg_deep_orange));
        //darkVibrant  -  有活力的暗色
        int darkVibrantColor = palette.getDarkVibrantColor(getResources().getColor(R.color.bg_deep_orange));
        //muted        -  柔和暗淡的颜色
        int mutedColor = palette.getMutedColor(getResources().getColor(R.color.bg_deep_orange));
        //lightMuted   -  柔和的亮色
        int lightMutedColor = palette.getLightMutedColor(getResources().getColor(R.color.bg_deep_orange));
        //darkMuted    -  柔和的暗色
        int darkMutedColor = palette.getDarkMutedColor(getResources().getColor(R.color.bg_deep_orange));

        imageView1.setBackgroundColor(vibrantColor);
        imageView2.setBackgroundColor(lightVibrantColor);
        imageView3.setBackgroundColor(darkVibrantColor);
        imageView4.setBackgroundColor(mutedColor);
        imageView5.setBackgroundColor(darkMutedColor);

使用第二种方式获取Pattle的颜色，也是经常采取的方法：

//获取颜色的第二种方法

//获取有活力颜色的采样对象 这里可以获取你想要的任何颜色类型的Swatch
Palette.Swatch vibrantSwatch = palette.getVibrantSwatch();

int bodyTextColor = vibrantSwatch.getBodyTextColor();//获取适合作为文字颜色
float[] hsl = vibrantSwatch.getHsl();//the HSL value of this color，即色相，饱和度，明度.
int population = vibrantSwatch.getPopulation();//获取明亮的颜色
int rgb = vibrantSwatch.getRgb();//获取适合作为背景RGB颜色
int titleTextColor = vibrantSwatch.getTitleTextColor();

//一般会将getRgb设置给控件背景色，getBodyTextColor()设置给文字颜色

textView.setBackgroundColor(rgb);
textView.setTextColor(bodyTextColor);

CardView的使用

CardLayout是一个拥有高度和阴影，以及轮廓裁剪，圆角的布局，在v7包中，本质继承自FrameLayout，所以要想在一个CardView中包含多个控件的需要使用另外的布局进行包裹，如LinearLayout；

添加依赖：
​
compile ‘com.android.support:cardview-v7:23.2.1’

各属性说明:

  1. 设置圆角：card_view:cardCornerRadius="10dp"
  2. 设置高度：card_view:cardElevation="10dp"
  3. 设置内边距：card_view:contentPadding="10dp"

  4. 设置背景色：card_view:cardBackgroundColor="?android:attr/colorPrimary"

    <android.support.v7.widget.CardView
    android:id="@+id/cardView"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    card_view:cardBackgroundColor="@color/button_normal_deep_purple"
    card_view:cardCornerRadius="30dp"
    card_view:cardElevation="20dp"
    card_view:contentPadding="10dp">

    <TextView
        android:layout_width="match_parent"
        android:layout_height="100dp"
        android:gravity="center"
        android:text="我是CardView"
        android:textColor="#fff"
        android:textSize="20sp" />
</android.support.v7.widget.CardView>

  <android.support.v7.widget.CardView
    android:id="@+id/cardView"
    android:layout_marginTop="20dp"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    card_view:cardBackgroundColor="@color/button_normal_blue_grey"

    card_view:cardElevation="20dp"
    card_view:contentPadding="10dp">
    <LinearLayout
        android:layout_width="match_parent"
        android:orientation="vertical"
        android:layout_height="wrap_content">

        <TextView
            android:background="@color/action_cyan"
            android:layout_width="match_parent"
            android:layout_height="100dp"
            android:gravity="center"
            android:text="我是CardView"
            android:textColor="#fff"
            android:textSize="20sp" />
        <View
            android:layout_marginTop="5dp"
            android:layout_marginBottom="5dp"
            android:layout_width="match_parent"
            android:layout_height="1dp"
            android:background="@color/action_cyan"
            />
        <TextView
            android:background="@color/action_pink"
            android:layout_width="match_parent"
            android:layout_height="100dp"
            android:gravity="center"
            android:text="我是CardView"
            android:textColor="#fff"
            android:textSize="20sp" />
    </LinearLayout>
</android.support.v7.widget.CardView>
